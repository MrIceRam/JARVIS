pygetwindow
Использование pygetwindow (более точный способ)
Позволяет свернуть текущее окно или переключиться на конкретное.
python
import pygetwindow as gw
# Свернуть все окна (аналог Win+D)
for window in gw.getAllWindows():
    window.minimize()
Примечание
Для использования этих библиотек их нужно установить:
bash
